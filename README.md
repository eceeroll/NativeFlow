# NativeFlow 

> **Learn English the way natives speak it** — one expression at a time.

NativeFlow is a React Native mobile app built with Expo that helps users learn natural English expressions through daily micro-learning sessions and AI-powered features. Instead of textbook English, NativeFlow teaches the idioms, phrasal verbs, slang, and conversational patterns that native speakers actually use.

---

## ✨ Features

- **Micro-learning sessions** — Short, focused daily lessons that fit into any schedule
- **AI-powered explanations** — Context-aware breakdowns of expressions, tone, and usage
- **Real conversation examples** — Every expression shown in authentic dialogue scenarios
- **Daily streaks & progress tracking** — Stay consistent with habit-building mechanics
- **Personalized learning path** — Adapts to your level and areas of focus
- **Offline support** — Core lessons available without an internet connection

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- iOS Simulator / Android Emulator, or the [Expo Go](https://expo.dev/go) app

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/nativeflow.git
cd nativeflow

# Install dependencies
npm install

# Start the development server
npx expo start
```

Then scan the QR code with **Expo Go** (Android) or the **Camera app** (iOS), or press `i` / `a` to open in a simulator.
